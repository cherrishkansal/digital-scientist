# Digital Scientist

Digital Scientist is an AI-powered research assistant that helps users:

* Generate research hypotheses
* Search academic papers
* Analyze datasets
* Generate statistical reports
* Create PDF research reports

## Features

* Hypothesis Generation
* arXiv Paper Search
* Evidence Analysis
* CSV Dataset Upload
* Correlation Analysis
* P-value Calculation
* Scatter Plot Generation
* PDF Report Generation
* Streamlit Dashboard

## Tech Stack

* Python
* FastAPI
* Streamlit
* Gemini API
* Pandas
* SciPy
* Matplotlib
* ReportLab

## How to Run

Run backend:

```bash
uvicorn main:app --reload
```

Run frontend:

```bash
streamlit run frontend.py
```

## Project Goal

To combine AI reasoning, academic research retrieval, statistical analysis, and report generation into a single research assistant platform.
