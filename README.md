# Digital Scientist

Digital Scientist is an AI-powered research assistant that converts natural language claims into formal hypotheses, searches academic papers, analyzes supporting and opposing evidence, and tests uploaded datasets statistically.

## Features

- Convert claims into research hypotheses
- Search academic papers
- Extract supporting and opposing evidence
- Analyze CSV datasets
- Generate scientific reports
- Provide confidence scores

## Tech Stack

- Python
- FastAPI
- OpenAI API
- arXiv API
- Pandas
- SciPy
- React
